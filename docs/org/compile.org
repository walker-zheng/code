* 编译链
** GNU
   autoconf autoscan automake autoheader
*** 基本生成过程   
   - autoscan生成configure.scan，编辑为configure.ac
   - autoconf生成configure
   - automake以Makefile.am生成Makefile.in
   - ./configure生成Makefile
   - make && make install
*** 其他     
   - autoheader生成config.h.in
   - automake --add-missing 添加扫描缺少的头文件
   - configure.ac中部分定义
     - AM_INIT_AUTOMAKE([1.12.5 foreign no-define])	//定义使用automake
     - AC_CONFIG_SRCDIR([src/apue_test.c])
